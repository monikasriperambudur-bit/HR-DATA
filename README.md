# HR ANALYTICS : End-to-End HR Data Analytics Dashboard using Power BI
## Project overview
The HR Analytics Dashboard is an interactive power bi project  designed to analyze employee data and generate meaningful insights related to staff performance,salary distribution,workforce demographics and organizational structure.
## Tech stack
The Dashboard was built using the following tools and technologies 
-**power bi desktop** - Dashboard development
-**power query** - Data cleaning and transformation
-**DAX** - Measures and kpis
-** Microsoft Excel/csv** - Dataset source
## Objective
The main obective of this project is to:
-Evaluate employee performance and workforce growth 
-Analyze salary distribution across job roles
-Identify top N
-Monitor leave balance 
## Data source
The dataset includes:
-Name
-Emp id
-Gender
-Education Qualification
-Date of join
-Job title
-Salary
-Age
-Leave balance
## Key Insights
-Gender diversity evaluation
-Employee performance overview
-Visibility into workforce growth and hiring trends
-Age demographic understanding
-Improved HR decision making through data visualization
